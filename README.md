# powerbi-ecommerce-sales-dashboard
Interactive Power BI dashboard analyzing e-commerce sales, profit trends, customer behavior, and payment insights with actionable business insights.

E-commerce Sales Dashboard (Power BI)
This project demonstrates my ability to analyze business data, build interactive dashboards, and generate actionable insights using Power BI.

Problem Statement
The objective of this project is to analyze e-commerce sales data to identify trends in revenue, customer behavior, product performance, and payment methods to support data-driven business decisions.

Dataset
E-commerce sales dataset including:
- Orders
- Customers
- Products
- Payment modes
- State-wise sales

Tools & Technologies
- Power BI
- Data Modeling

- DAX (Data Analysis Expressions)
- Total Sales = SUM(Sales[Amount])
- Total Profit = SUM(Sales[Profit])
- Profit Margin = DIVIDE([Total Profit], [Total Sales])

Dashboard Features
- Monthly profit trends analysis
- Category-wise sales distribution
- Payment mode analysis
- State-wise revenue breakdown
- Top customers identification
- Sub-category profit analysis

Key Insights
- Clothing dominates sales with 63% contribution
- Cash on Delivery (COD) is the most preferred payment method (44%)
- Maharashtra generates the highest sales revenue
- Top-performing sub-categories include printers, bookcases, and sarees

Business Impact
- Helps identify high-performing product categories
- Enables targeted marketing strategies
- Improves payment strategy optimization
- Supports regional sales planning

Business Recommendations
- Focus marketing efforts on the Clothing category (63% contribution)
- Promote digital payments to reduce dependency on COD (44%)
- Target high-revenue states like Maharashtra for expansion
